迁移Win10的Linux子系统

2019-04-24
笔记

问题

Win10 安装的 Linux 子系统默认安装在 C 盘，对于需要在 Linux 系统中使用大文件的用户来说不方便。下面记录一种方法可以将 Linux 文件系统迁移出 C 盘。

如果要查看 Linux 安装目录， 在文件管理器地址栏输入 %localappdata%\Packages，找到一个文件夹 以 CanonicalGroupLimited.UbuntuonWindows 开头的，就是 Linux 安装目录。Linux 文件在该目录的 LocalState\rootfs

解决

  1. 查看当前用户名，我的用户名是 lingl：

  2. 在迁移目标磁盘创建一个文件夹 ，并设置权限。
    例如创建 E:\Ubuntu.windows 文件夹后，执行如下命令（lingl 替换成自己的用户名）:

  3. 下载 LxRunOffline（点此下载）。

  4. lxrunoffline list 查看已安装的 Linux 发行版本：

  5. lxrunoffline move 迁移 Linux 文件：

  6. 确认迁移完成，使用 lxrunoffline get-dir -n Ubuntu 查看安装目录。

  7. 可以删除原 LocalState\rootfs 目录。
