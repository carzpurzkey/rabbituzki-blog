使用管理员权限运行 Windows Terminal 并添加“在此打开 Windows Terminal”至上下文菜单

¶使用管理员权限运行 Windows Terminal

  1. 新建一个快捷方式。

  2. 在快捷方式的“位置”处填写C:\Windows\system32\cmd.exe /c start /b wt。

    Windows Termianl 的可执行程序的软件连接一般位于 %localappdata%\Microsoft\WindowsApps\wt.exe

  3. 打开快捷方式的“属性”对话框，选择“高级”，选择“以管理员身份运行”。

¶添加“在此打开 Windows Terminal”至上下文菜单

  1. 设置菜单项的图标：

    • (1) 在注册表_HKEY_CLASSES_ROOT\Directory\Background\shell_ 下新建名为“在此打开 Windows Terminal”的项。
    • (2) 在 (1) 所建项下新建名为“Icon”的 REG_SZ 值，数值数据为图标文件或带图标的可执行文件完整目录。
