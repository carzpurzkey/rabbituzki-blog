¶hexo 的基本命令

  • hexo init [folder] ：创建一个存放网站源码的文件夹。
  • hexo new [layout] <title>：新建一篇文章。
    layout：默认 post，可以是 page、draft。
    title：文章标题，标题包含空格的话，请使用引号括起来。
  • hexo clean：清除缓存文件 db.json 和已生成的静态文件 public,尤其是更换主题后
  • hexo g：生成静态 html 文件
  • hexo s：启动服务器
  • hexo d：部署网站
