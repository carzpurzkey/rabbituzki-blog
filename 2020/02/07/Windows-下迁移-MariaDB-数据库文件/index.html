Windows 下迁移 MariaDB 数据库文件

  1. 运行 MySQL Client (MariaDB 10.4 (x64))，输入密码后登录。

  2. 执行语句 show global variables like "%datadir%";，查询结果中的 Value 列所示即为 MariaDB 数据库文件存储位置。

  3. 停止 MariaDB 服务。

  4. 进入上一步数据库文件存储路径，将以数据库名命名的文件夹及 mysql 文件夹和以 ib 开头的文件复制至将要迁移的目标目录。

  5. (可选) 删除原始目录中上一步相应文件。

  6. 启动数据库即完成迁移。
